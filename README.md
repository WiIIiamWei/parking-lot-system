# 停车场管理系统

### 介绍
NJUPT程序设计（上机）课设项目，基于Python-PyQt5实现GUI。

新手的尝试，大佬勿喷（

### 使用说明
使用 `git` 克隆本仓库：
```
git clone https://gitee.com/WiIIiamWei/parking_lot_system.git
```
或直接下载源代码并运行Python文件。

*麻烦老师给点高分 (>_<)*

### 已经实现的功能

- 用户/管理员注册
- 权限区分
- 停车场GUI画面
- 用户停车计时收费
- 管理员停车场管理
